<h2>Project Overview</h2>

<p>
This project builds an end-to-end <b>Sales Analytics Platform</b> that automatically converts raw sales datasets into actionable business insights using an <b>Automated ETL Pipeline</b> and an interactive dashboard.
</p>

<p>
Users can upload raw datasets and the system automatically processes them to generate analytics including:
</p>

<ul>
<li>Revenue Trends</li>
<li>Customer Segmentation</li>
<li>Product Performance</li>
<li>Predictive Insights</li>
</ul>

<hr>

<h2>Problem Statement</h2>

<p>
Many businesses rely on spreadsheets for analyzing sales data which leads to several problems:
</p>

<ul>
<li>Manual data cleaning</li>
<li>Inconsistent reporting formats</li>
<li>Lack of real-time analytics</li>
<li>Poor visibility into customer behaviour</li>
<li>Difficulty identifying product trends</li>
</ul>

<p>
This project solves the problem by building an automated ETL pipeline integrated with an analytics dashboard.
</p>

<hr>

<h2>Objectives</h2>

<ul>
<li>Automate data ingestion and transformation</li>
<li>Generate analytics from raw sales datasets</li>
<li>Create real-time dashboards</li>
<li>Analyze customer behaviour</li>
<li>Generate product performance insights</li>
<li>Provide predictive business insights</li>
</ul>

<hr>

<h2>System Workflow</h2>

<h2>Installation</h2>

<pre>
git clone https://github.com/Dhruva912005/Sales-Analytics-Dashboard-with-Automated-ETL-Pipeline.git

cd Sales-Analytics-Dashboard-with-Automated-ETL-Pipeline

pip install -r requirements.txt

python app.py
</pre>

<p>Open browser:</p>

<pre>
http://localhost:5000
</pre>
