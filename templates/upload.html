Requirements & Guidelines
Required Columns:
  • order_id - Unique order identifier
  • order_date - Date (YYYY-MM-DD format)
  • customer_name - Customer's name
  • product_name - Product name
  • quantity - Number of units (numeric)
  • unit_price - Price per unit (numeric)
  • total_price - Total order value (numeric)
Optional Columns:
  • category - Product category
  • country - Customer country
  • payment_mode - Payment method
Notes:
  • Column names are automatically detected
  • Missing columns will be created with defaults
  • Duplicate orders are automatically skipped
  • Maximum file size: 16MB
